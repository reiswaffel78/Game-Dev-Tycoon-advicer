Welcome to the Game Dev Tycoon Wiki. [Game Dev Tycoon](https://gamedevtycoon.fandom.com/wiki/Game_Dev_Tycoon "Game Dev Tycoon") was created by Greenheart Games, which was founded in July 2012, by brothers Patrick and Daniel Klug with an agenda to develop games that are "fun to play instead of mind numbing money grabbers. Less social, less ville, more game is our motto."

"Our first game to test the waters is Game Dev Tycoon, a tycoon style game where you can start your own game development company in the 80s. Game Dev Tycoon was inspired by [Game Dev Story](https://kairosoft.fandom.com/wiki/Game_Dev_Story "w:c:kairosoft:Game Dev Story") (by Kairosoft), which was the first ‘tycoon’ game we enjoyed playing on the iPhone; however, from the start, we wished the game would work and look differently. We wanted a game development simulation which would be less random, more about your choices and a little more realistic. We began working on what would become Game Dev Tycoon in 2011 and created the concepts and game mechanics as well as the game itself from scratch."

## Walkthrough

[Game Development](https://gamedevtycoon.fandom.com/wiki/Game_Development "Game Development") provides advice on how to develop games, including known good Topic/Genre combos, broad slider guidelines for each genre & multi genre, what consoles work with what genre etc. If you prefer an exhaustive walkthrough for the whole game, [Success Guide](https://gamedevtycoon.fandom.com/wiki/Success_Guide "Success Guide") aims to provide such help.

## Game Knowledge

There is a substantial amount of knowledge about the game scattered around: If you simply want some unlocking information, check [Research Unlocks](https://gamedevtycoon.fandom.com/wiki/Research_Unlocks "Research Unlocks"), or [Random Events](https://gamedevtycoon.fandom.com/wiki/Random_Events "Random Events"), [Custom Game Engine](https://gamedevtycoon.fandom.com/wiki/Custom_Game_Engine "Custom Game Engine"), [Achievements](https://gamedevtycoon.fandom.com/wiki/Achievements "Achievements"), [R&D lab](https://gamedevtycoon.fandom.com/wiki/R%26D_lab "R&D lab"), [Convention](https://gamedevtycoon.fandom.com/wiki/Convention "Convention"), [Training](https://gamedevtycoon.fandom.com/wiki/Training "Training").

If you are on the technical side of things, you can better understand how the game works thanks to our in-depth explanations of the most important algorithms: [Review Algorithm](https://gamedevtycoon.fandom.com/wiki/Review_Algorithm "Review Algorithm"), [Tech and Design Points Generation Algorithm](https://gamedevtycoon.fandom.com/wiki/Tech_and_Design_Points_Generation_Algorithm "Tech and Design Points Generation Algorithm"), [Sales Algorithm](https://gamedevtycoon.fandom.com/wiki/Sales_Algorithm "Sales Algorithm"), [Staff](https://gamedevtycoon.fandom.com/wiki/Staff "Staff"). You will also need to see [Modding](https://gamedevtycoon.fandom.com/wiki/Modding "Modding") to access all the assets of the game. You can also find nicely formatted [Raw Data](https://gamedevtycoon.fandom.com/wiki/Raw_Data_for_Review_Algorithm "Raw Data for Review Algorithm"), explanation for various [Game Flags](https://gamedevtycoon.fandom.com/wiki/Game_Flags "Game Flags"), and some more [Raw Data](https://gamedevtycoon.fandom.com/wiki/Raw_Data_for_Sales_Algorithm "Raw Data for Sales Algorithm")
