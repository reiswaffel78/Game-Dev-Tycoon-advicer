# Game Dev Tycoon Advisor

> Your optimal setup companion for Game Dev Tycoon (including Netflix edition)

Game Dev Tycoon Advisor is a free, community-driven web application that helps players of the game "Game Dev Tycoon" find the best topic-genre combinations, platform recommendations, slider presets, and complete game plans. All data is sourced from community research with trust-weighted conflict resolution.

## Website

- URL: https://gamedevtycoonadvisor.com
- Languages: English (default), German, French, Italian, Spanish, Korean, Japanese, Chinese, Hindi, Turkish, Portuguese, Russian, Czech, Dutch, Arabic, Greek, Hungarian, Polish, Swedish

## Tools

### Dashboard
- Path: /
- Description: Overview with quick stats, game tips, and navigation to all tools and guides.

### Topic Recommender
- Path: /recommend/topic
- Description: Find the best topic for a chosen genre. Shows scored topic-genre fit combinations ranked by compatibility.

### Genre Recommender
- Path: /recommend/genre
- Description: Find the best genre for a chosen topic. Shows scored genre-topic fit combinations ranked by compatibility.

### Platform Recommender
- Path: /recommend/platform
- Description: Find the best platform for a chosen genre and target audience. Shows platform-genre and platform-audience fit scores.

### Slider Presets
- Path: /sliders
- Description: Optimal development slider settings (Engine, Gameplay, Story/Quests, Dialogues, Level Design, AI, World Design, Graphics, Sound) for every genre, broken down by development stage.

### Game Planner
- Path: /planner
- Description: Full game planning tool. Input your current game state (year, cash, fans) and get a prioritized list of recommended game releases with optimal topic+genre+platform+audience combinations and research priorities. Supports filtering by unlocked topics.

## API

All data endpoints are available under `/api/*`:

- GET /api/topics - List all game topics
- GET /api/genres - List all game genres
- GET /api/platforms - List all platforms with unlock years and costs
- GET /api/audiences - List all target audiences
- GET /api/topic-genre-fits - Topic-genre compatibility scores
- GET /api/platform-genre-fits - Platform-genre compatibility scores
- GET /api/platform-audience-fits - Platform-audience compatibility scores
- GET /api/development-weights - Slider weights per genre and stage
- POST /api/recommend/by-topic - Get genre recommendations for a topic
- POST /api/recommend/by-genre - Get topic recommendations for a genre
- POST /api/recommend/by-platform - Get platform recommendations
- POST /api/planner - Generate a full game plan based on current state

## i18n URL Structure

- English (default): base paths (e.g., `/sliders`, `/planner`)
- Other languages: path with language suffix and trailing slash (e.g., `/sliders/de/`, `/planner/ja/`, `/faq/fr/`)
- All pages support 19 languages with hreflang alternate links
